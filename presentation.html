возможности

Что умеет

🔑

Автопароли ZIP

Зетта и Сбер: сам находит пароли в отдельных письмах и распаковывает архивы

🧹

Дедупликация

По ФИО + полис + даты. Нормализация дат. Не будет двойных записей

📧

Отчёт на почту

HTML-отчёт + xlsx + csv. Только новые записи за текущий запуск

📁

Экспорт для 1С

Ежедневный CSV на сетевой диск (SMB). Автоподхват в 1С

🛡️

Безопасность

Env vars для паролей, защита от formula injection, file locking, аудит-лог

❤️

Мониторинг

Healthcheck-пинг, карантин ошибок, бэкапы master.xlsx перед каждой записью
